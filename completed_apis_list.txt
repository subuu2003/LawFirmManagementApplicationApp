## Authentication APIs
1. POST /api/auth/login_username_password/ - Login with username/email and password.
2. POST /api/auth/request_phone_otp/ - Request OTP for phone-based login.
3. POST /api/auth/request_email_otp/ - Request OTP for email-based login.
4. POST /api/auth/verify_otp/ - Verify OTP and obtain authentication token.
5. POST /api/auth/set_password/ - Set initial password for users added by admins.
6. POST /api/auth/logout/ - Invalidate token and log out.
